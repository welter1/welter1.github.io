Projects




Robot Hand (Intro to Engineering Team Project)

The assignment was to use an Arduino board to design anything we wanted and present it at the end of the semester, the only requirement was that it functioned. The professor told the class that the robot hand had been attempted every year but hadn’t been completed by anyone in the freshman intro to engineering class. My team and I figured that was due to teams over complicating the product, so we gave it a shot.
We started simply with a toy hand we found online. From there we broke it down and connected the fingers to servo motors using fishing line. Intending to reduce complexity we decided to use Styrofoam for the interior structure due to the fact it was easy to mold and also easy to replace in case of errors in the build. Once we had mapped out the placement of the servos and built our mold, we connected the flex sensors to our glove and adjusted our code to have the robot hand show Realtime reflection of the user’s movements. After we had completed the project and presented, the total cost of the robot hand was 11 dollars. (not including the Arduino kits that were given to us in class) which turned out to be 1/4th the cost of the average previous attempts before us. I learned a lot from this project but most importantly how necessary a good team is when it comes to designing, developing, and implementing a project.





(IMDB) Database Project



Group Database project to pick an online Database and recreate it, my team and I picked IMDB, we created a database from scratch, starting by laying out our ER and EER diagrams, we then converted the ER diagram to a Relational Database Schema and implemented the data using oracle DDL language. Once the database was complete, we developed queries in Oracle DML to show the functionality of our database. I've included the ER diagram and the Relational Database Schema below. I will be uploading the database to github within a few weeks for anyone intrested in seeing the completed product.
